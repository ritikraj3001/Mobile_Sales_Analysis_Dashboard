# Mobile Sales Analysis Dashboard

## 📌 Project Overview

The **Mobile Sales Analysis Dashboard** is an interactive Power BI project that provides valuable insights into mobile phone sales performance across different cities, brands, payment methods, and time periods. The dashboard enables stakeholders to monitor key business metrics, identify sales trends, and make data-driven decisions.

---

## 🎯 Project Objectives

- Analyze overall mobile sales performance.
- Track sales across different cities in India.
- Identify the best-selling mobile models.
- Understand customer payment preferences.
- Monitor monthly sales trends.
- Evaluate brand-wise sales performance.
- Analyze customer ratings.
- Generate actionable business insights.

### 💳 Payment Method Analysis
